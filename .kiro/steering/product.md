# Product Overview

This is a personal portfolio website for Ivan Tanaka, a software engineer with 8+ years of experience.

## Purpose
- Showcase professional experience, skills, and projects
- Display certifications and education
- Provide contact information for potential collaborations
- Support PDF export/print functionality for resume format

## Key Features
- Single-page resume/portfolio layout
- Dark theme with amber accent colors
- Print-optimized styling (converts to light theme)
- Dynamic content loaded from JSON data files
- Portfolio project showcase with external links
- Certifications with verification links
- Contact section with social links
- Google Analytics integration

## Target Audience
Recruiters, potential clients, and collaborators looking to learn about Ivan's professional background and work.
